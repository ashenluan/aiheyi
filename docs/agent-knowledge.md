# 飞彩工作室 — AI 助手知识库

> 供 FC 智能体（AgentFAB）系统提示词使用的结构化知识库。
> 按页面分区组织，支持按 pathname 动态注入。

---

## 全局知识（所有页面可用）

### 软件简介
飞彩工作室是一款 AI 视频分镜创作工具，核心功能：
- **生图工作台**（/studio）：九宫格/四宫格/自定义宫格分镜生图
- **图生视频**（/video）：使用即梦 Seedance 2.0 / 即梦 3.x 将分镜图片生成视频
- **分镜流水线**（/pipeline）：节拍拆解 → 智能分镜 → 智能体分镜
- **剧本管理**（/scripts）：导入/管理剧本（支持 TXT/CSV/粘贴）
- **即梦生图**（/jimeng）：使用即梦模型直接生成图片
- **设置**（/settings）：LLM API 配置、即梦账户、输出路径

### 常见操作流程
1. **新手上路**：设置页配置 LLM API Key → 剧本管理导入剧本 → 流水线分析 → 工作台生图 → 生成视频
2. **快速出图**：工作台直接填写提示词 → 选择生图引擎（Gemini/即梦/API）→ 点击生成
3. **一致性维护**：左侧面板「一致性」标签 → AI 一键提取（角色/场景/道具）→ 绑定参考图 → 生图时自动拼接

### 设置页配置指南
- **LLM API Key**：必填。支持 OpenAI 兼容格式的所有模型（通义千问、DeepSeek、GPT、Claude 等）
- **API 预设**：点击预设按钮可一键填入常用配置（硅基流动、通义千问等）
- **即梦账户**：Seedance 视频生成需要即梦 Cookie，系统会通过浏览器代理自动获取
- **输出目录**：默认 outputs/，可在 feicai-paths.json 中配置

### 支持的生图引擎
| 引擎 | 说明 | 使用方式 |
|------|------|---------|
| Gemini | Google Gemini 网页自动化生图 | 需启动 Gemini Tab 服务 |
| 即梦 | 字节跳动即梦 API 生图 | 需登录即梦账户 |
| API | OpenAI 图像生成协议（支持七牛云等） | 设置页配置图像 API |

### 支持的视频模型
| 模型 | 标识 | 积分/秒 |
|------|------|--------|
| Seedance 2.0 Pro | dreamina_seedance_40_pro | 720P=5, 1080P=8 |
| Seedance 2.0 Fast | dreamina_seedance_40 | 720P=3, 1080P=5 |
| 即梦 3.5 Pro | video-3.5-pro | 8 |
| 即梦 3.0 Pro | video-3.0-pro | 10 |
| 即梦 3.0 | video-3.0 | 2 |
| 即梦 3.0 Fast | video-3.0-fast | 2 |

## /studio — 生图工作台

### 页面功能
- 九宫格/四宫格/智能分镜/自定义宫格四种模式
- 每个格子可独立编辑提示词、生图、超分、下载
- 左侧面板：提示词列表、一致性管理、角色库
- 支持从流水线加载提示词、从智能体推送分镜

### 常见操作
- **切换宫格模式**：顶部模式选择器（九宫格/四宫格/智能分镜/自定义）
- **生成图片**：填入提示词后点击格子上的生成按钮，或使用底部「全部生成」
- **超分辨率**：点击格子上的放大图标可将图片 2x 放大
- **绑定参考图**：右键格子 → 绑定参考图，生图时会自动拼接
- **切换集数**：顶部 EP 选择器切换不同集数的提示词
- **导入宫格图**：支持从外部导入合成图并自动切分为 4/9 格
- **切换生图引擎**：左上角引擎切换按钮（Gemini/即梦/API）

### 常见问题
| 问题 | 解决 |
|------|------|
| 图片生成后不显示 | 检查生图引擎是否正确配置；F12 Console 查看错误 |
| 提示词为空 | 先通过流水线生成提示词，或手动填写 |
| 刷新后图片消失 | 图片已保存到 outputs/grid-images/ 磁盘目录，重新打开会自动恢复 |
| 超分失败 | 检查原图是否已生成；确认 API 配置正确 |

## /video — 图生视频

### 页面功能
- 从宫格导入分镜图片 → 选择视频模型 → 生成视频
- 支持 Seedance 2.0（Pro/Fast）和即梦 3.x（4种模型）
- 可设置时长（4-15秒）、画质（720P/1080P）、宽高比

### 常见操作
- **导入图片**：点击「从宫格导入」选择要生成视频的分镜格
- **选择模型**：Seedance 2.0 Pro（高质量）、Fast（快速）、即梦 3.x（多种选择）
- **提示词**：可为每个视频片段添加动态描述提示词
- **批量生成**：选中多个片段后一键批量提交

### 常见问题
| 问题 | 解决 |
|------|------|
| 积分不足 (ret=5000) | 去即梦官网 jimeng.jianying.com 领取/购买积分 |
| 内容被过滤 (fail_code=2038) | 修改提示词避免违规内容 |
| 浏览器连接失败 | 检查设置页 → 尝试「重新连接浏览器」；或检查 Playwright 是否安装 |
| 未获取到记录ID | 重新登录即梦网站获取新 Cookie |

## /pipeline — 分镜流水线

### 页面功能
- **节拍拆解**：上传剧本 → AI 分析叙事节拍 → 每集关键情节点
- **智能分镜**：基于节拍自动生成分镜画面描述和英文提示词
- **自定义分镜**：AI 对话交互式分镜创作（3角色 Agent）

### 常见操作
- **分析剧本**：选择剧本 → 点击「分析」→ AI 自动拆解章节和节拍
- **锁定节拍**：确认节拍后点击锁定，进入分镜生成
- **加载到工作台**：生成的提示词可一键加载到生图工作台

### 常见问题
| 问题 | 解决 |
|------|------|
| 分析超时 | 剧本太长时可选择单章节分析；或换用更快的模型 |
| 节拍数量不对 | 可手动调整节拍数量和内容 |
| 提示词质量不高 | 在工作台中逐个编辑优化，或使用 AI 改写 |

## 常见报错速查

### LLM API 报错
| 报错 | 原因 | 解决 |
|------|------|------|
| API 返回 401/403 | API Key 无效或过期 | 设置页检查/更换 API Key |
| API 返回 429 | 请求限流 | 等 1-2 分钟重试；系统已内置自动重试 |
| API 返回 500/502/503 | 服务端错误 | 换 API 预设或稍后重试 |
| 未配置 LLM API Key | 设置页未填写 | 进入设置页 → 填写 API Key |
| CDN 暂时不可用 | API 代理返回 HTML | 切换到国内直连 API 预设 |

### 即梦/Seedance 报错
| 报错 | 原因 | 解决 |
|------|------|------|
| 积分不足 (ret=5000) | 账户积分余额不足 | 在即梦官网充值 |
| 内容被过滤 (2038) | 安全审核拦截 | 修改提示词避免违规描述 |
| 权限拒绝 (ret=3018) | Cookie 过期/不完整 | 重新登录即梦网站 |
| 人脸检测 (ret=4010) | 人脸安全检测触发 | 在即梦完成人脸授权 |
| 请求被拒绝 (ret=1000) | 请求参数错误 | 升级到最新版本 |
| 浏览器连接失败 | Playwright 未安装 | 首次使用 Seedance 功能时会自动安装；如失败，手动点击设置页「安装浏览器」 |

### 生图报错
| 报错 | 原因 | 解决 |
|------|------|------|
| 无法连接 Gemini Tab | 服务未启动 | 检查 Gemini Tab 页面是否已启动 |
| 图像 API 频率限制 | 429 限流 | 等 1-2 分钟后重试 |
| 图像生成失败 | API 服务端错误 | 检查图像 API 配置；换用其他生图引擎 |

### 通用问题
| 问题 | 解决 |
|------|------|
| 页面白屏/卡死 | F5 刷新页面；清除浏览器缓存 |
| 操作后数据丢失 | 系统已自动保存到磁盘，刷新后会恢复 |
| 热更新失败 | 检查网络连接 → 重启软件重试 → 或下载最新完整包 |
| Mac 启动失败 | 确保已安装 Node.js（通过 install.command 自动安装） |

## 快捷技巧

1. **AI 助手可以做什么**：生成分镜、修改提示词、批量操作、导航页面、切换模式——直接用自然语言告诉我
2. **快速跳转**：说"打开工作台"、"去设置页"、"打开流水线"可直接跳转
3. **批量操作**：说"全部翻译"、"全部超分"、"批量生成 1,3,5 格"
4. **一致性参考**：在工作台左侧「一致性」面板绑定角色/场景参考图，生图时自动保持风格统一
5. **多集管理**：通过 EP 选择器切换集数，每集的提示词独立保存
6. **角色库**：收藏常用角色到角色库，跨项目复用

## 高级工作流

### 从零到成片的完整流程
1. **设置 API**：设置页配置 LLM API Key（推荐硅基流动或通义千问）
2. **导入剧本**：剧本管理页导入 TXT/CSV 文件
3. **AI 分析**：流水线页→节拍拆解→AI 分析关键情节点
4. **生成分镜**：智能分镜自动生成画面描述和英文提示词，或使用自定义分镜由 AI 对话式创作
5. **加载到工作台**：将分镜提示词加载到生图工作台
6. **一致性设置**：AI 提取角色/场景/道具 → 绑定参考图
7. **批量生图**：选择生图引擎（Gemini/即梦/API）→ 全部生成
8. **超分优化**：对关键画面进行超分辨率放大
9. **生成视频**：导入分镜图到视频页 → 选择 Seedance/即梦模型 → 生成

### 高效批量操作技巧
- **一句话改风格**：告诉 AI "所有格子改成赛博朋克风格"→ 自动批量改写
- **一键翻译**：生成中文描述后说"全部翻译"→ 自动翻译为英文提示词
- **选择性生图**：说"生成第 1、3、5 格的图片"→ 只生成指定格子
- **分集管理**：说"分成3集，每集9格"→ AI 自动按叙事逻辑拆分

### 专业创作技巧
- **景别节奏**：好的分镜应该有景别变化（远景建立→中景叙事→特写情感→远景收束）
- **色彩叙事**：开头暖色调→中段冷色调→高潮对比色→结尾回归暖色，用色彩表达情感变化
- **留白设计**：不要每格都塞满信息，适当留白让观众思考
- **视线引导**：同一场景中角色视线方向应该保持连贯（正反打遵循180度规则）
- **节奏对比**：静态画面后接动态画面，安静场景后接激烈场景，制造张力
