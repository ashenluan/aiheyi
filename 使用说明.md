# FEICAI Studio - AI智能分镜系统 V2.16

## 使用方法

1. 双击 FEICAI-Studio.exe（或 启动FEICAI-Studio.bat）即可启动
2. 浏览器会自动打开 http://localhost:3000
3. 首次使用请在「设置」页面配置您的 API Key 和 Base URL
4. 关闭命令行窗口即可停止服务

## 配置说明

在「设置」页面需要配置:
- LLM API Key 和 Base URL（用于剧本分析和提示词生成）
- 图像 API Key 和 Base URL（用于分镜图生成）
- 视频 API Key 和 Base URL（用于视频生成，可选）

支持 OpenAI 兼容的 API 接口。
